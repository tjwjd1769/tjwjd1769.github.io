# 임서정 · AMD 포트폴리오

이커머스 AMD(어소시에이트 MD) 경력 포트폴리오 웹사이트입니다.

## 보는 방법

1. **브라우저에서 바로 열기**  
   프로젝트 폴더에서 `index.html` 파일을 더블클릭하거나, 브라우저 주소창에 아래처럼 넣어서 열 수 있습니다.
   ```
   file:///Users/kimyanghoon/Desktop/dev/seojung/index.html
   ```

2. **로컬 서버로 열기 (선택)**  
   터미널에서 프로젝트 폴더로 이동한 뒤 다음 중 하나를 실행하세요.
   ```bash
   cd /Users/kimyanghoon/Desktop/dev/seojung
   python3 -m http.server 8080
   ```
   브라우저에서 `http://localhost:8080` 으로 접속합니다.

## 구성

- **About** – AMD·이커머스 경험 요약
- **Career** – 몬트그레이, 롯데홈쇼핑, 비엠스마일 경력 타임라인
- **가치관 & 강점** – 운영 안정성, 흐름 연결, 신뢰/책임
- **Skills** – 업무 도구·역량
- **학력·자격증** – 학력 및 자격증
- **Contact** – 연락처, 희망직무(MD > AMD)

이력서 PDF를 바탕으로 작성되었으며, 수정이 필요하면 `index.html`과 `style.css`를 편집하면 됩니다.
